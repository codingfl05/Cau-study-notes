Stack (스택)

Definition (정의)

LIFO — Last-In First-Out

Array Implementation (배열로 구현)

typedef struct { int top; int capacity; int *array; // char, int, float, double... } Stack; Stack *createStack(int capacity) { Stack *stack = (Stack*) malloc(sizeof(Stack)); if (!stack) return NULL; stack->top = -1; // 빈 스택: top = -1 stack->capacity = capacity; stack->array = (int*) malloc(capacity * sizeof(int)); if (!stack->array) return NULL; return stack; } int isFull(Stack *s) { return s->top == s->capacity - 1; } int isEmpty(Stack *s) { return s->top == -1; } int peek(Stack *s) { return s->array[s->top]; } void push(Stack *s, int item) { if (!isFull(s)) s->array[++s->top] = item; // 먼저 top 증가! else printf("Stack is full!\n"); } int pop(Stack *s) { if (!isEmpty(s)) return s->array[s->top--]; // 반환 후 top 감소! printf("Stack is Empty!\n"); return 0; }
push: ++top 먼저 증가  ·  pop: top-- 반환 후 감소  ·  top = -1이 빈 스택!

응용 1 — Palindrome 체크 (회문 검사)

회문: 앞뒤로 읽어도 같은 단어. 예: mom, level, radar, ABCDCBA

알고리즘:
① 앞 절반(length/2)을 스택에 push
② 홀수 길이면 가운데 문자 skip
③ 나머지 절반과 pop 결과 비교 → 다르면 Not Palindrome

int isPalindrome(Stack *stack, char word[], int length) { // 1. Push first half into stack for (int i = 0; i < length / 2; i++) push(stack, word[i]); int i = length / 2; if (length % 2 != 0) i++; // 홀수: 중간 문자 skip // 2. Pop and compare with second half while (i < length) { if (pop(stack) != word[i]) return 0; // Not palindrome i++; } return 1; // Palindrome! }

Array Implementation (배열로 구현)

typedef struct { int front, rear; int capacity; int *array; } Queue; Queue *createQueue(int capacity) { Queue *queue = (Queue*) malloc(sizeof(Queue)); if (!queue) return NULL; queue->front = 0; queue->rear = -1; // 초기 상태: rear=-1, front=0 queue->capacity = capacity; queue->array = (int*) malloc(capacity * sizeof(int)); if (!queue->array) return NULL; return queue; } int isFull(Queue *q) { return q->rear == q->capacity - 1; } int isEmpty(Queue *q) { return q->front == q->rear; } // front tracks head void enqueue(Queue *q, int item) { if (!isFull(q)) q->array[++q->rear] = item; // rear 먼저 증가! else printf("Queue is Full!\n"); } int dequeue(Queue *q) { if (!isEmpty(q)) return q->array[q->front++]; // front 반환 후 증가! printf("Queue is Empty!\n"); return -1; }
enqueue: ++rear 먼저 증가  ·  dequeue: front++ 반환 후 증가  ·  front==rear가 빈 큐!

⚠️ Linear Queue의 한계

아래 상황을 생각해보자. capacity=5인 큐에서 여러 번 enqueue/dequeue 후 rear가 배열 끝에 도달했다. front 앞에 공간이 남아있어도 큐가 "full"로 판단된다!

Circular Queue (원형 큐)

배열을 원형으로 사용하여 앞에 생긴 빈 공간을 재활용. % capacity 연산으로 인덱스를 순환시킨다.

// 핵심: % capacity 로 인덱스 순환 void enqueue(Queue *q, int item) { if (!isFull(q)) { q->rear = (q->rear + 1) % q->capacity; // 순환! q->array[q->rear] = item; q->size += 1; } else printf("Queue is Full!\n"); } int dequeue(Queue *q) { if (!isEmpty(q)) { int element = q->array[q->front]; q->front = (q->front + 1) % q->capacity; // 순환! q->size -= 1; return element; } printf("Queue is Empty!\n"); return -1; } // Circular Queue는 size 필드로 empty/full 판단! int isFull(Queue *q) { return q->size == q->capacity; } int isEmpty(Queue *q) { return q->size == 0; }
핵심 공식: (index + 1) % capacity → 배열 끝에 도달하면 자동으로 앞으로 돌아옴!

Traversal & Search (순회 & 검색)

// Traversal: 모든 노드 방문 void traverse(Node *head) { Node *cur_node = head; while (cur_node != NULL) { printf("%d ", cur_node->element); cur_node = cur_node->next; // 다음 노드로 이동 } } // Search (Iterative): key 값 찾기 bool search_iterative(Node *head, int key) { Node *cur_node = head; while (cur_node != NULL) { if (cur_node->element == key) return true; cur_node = cur_node->next; } return false; // key not found } // Search (Recursive): 재귀로 탐색 bool search_recursive(Node *head, int key) { if (head == NULL) return false; // Base case if (head->element == key) return true; // Found! return search_recursive(head->next, key); // Recurse }
종료 조건: cur_node != NULL — NULL을 만나면 리스트 끝!

Deletion (삭제) — 3가지 경우

① 첫 번째 노드 삭제

(1)temp = *head_ref
(2)*head_ref = head→next
(3)free(temp)

② 마지막 노드 삭제

(1)끝까지 traverse
(prev, cur)
(2)temp = cur_node
(3)prev→next = NULL
(4)free(temp)

③ 중간 노드 삭제

(1)위치 탐색
(prev, cur)
(2)temp = cur_node
(3)prev→next = cur→next
(4)free(temp)
void deleteNode(Node **head_ref, int key) { if (*head_ref == NULL) { printf("List is Empty.\n"); return; } // Case 1: 첫 번째 노드 삭제 if ((*head_ref)->element == key) { Node *temp = *head_ref; *head_ref = (*head_ref)->next; free(temp); return; } // Case 2 & 3: 중간 또는 마지막 노드 삭제 Node *prev_node = *head_ref; Node *cur_node = (*head_ref)->next; while (cur_node->next != NULL && cur_node->element != key) { prev_node = cur_node; cur_node = cur_node->next; } if (cur_node->element != key) { printf("Deletion failed: No key in the list.\n"); return; } Node *temp = cur_node; prev_node->next = cur_node->next; // 포인터 우회 free(temp); // 메모리 해제! } // 전체 리스트 삭제 void deleteList(Node **head_ref) { Node *cur_node = *head_ref; while (cur_node != NULL) { Node *next_node = cur_node->next; // 1. 다음 주소 저장 free(cur_node); // 2. 현재 노드 해제 cur_node = next_node; // 3. 다음으로 이동 } *head_ref = NULL; }
삭제 시 반드시 free()로 메모리 해제! 안 하면 Memory Leak!
