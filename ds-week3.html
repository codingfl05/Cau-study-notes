Array (배열)

Definition (정의)

연속된 메모리 공간에 저장된, 고정 크기의 동질(homogeneous) 요소 모음.

특성설명

Multidimensional Array (다차원 배열)

// 2D Array: 2행 3열 int arr2[2][3] = { {10,20,30}, {40,50,60} }; // 3D Array: 2×2×2 int arr3[2][2][2] = { {{10,20},{30,40}}, {{50,60},{70,80}} }; // 2D 순회 출력 for (int i = 0; i < 2; i++) { for (int j = 0; j < 3; j++) { printf("%d ", arr2[i][j]); } printf("\n"); }

Passing Array to Function (함수에 배열 전달)

배열 이름은 자동으로 첫 번째 요소의 포인터가 된다. arr == &arr[0]

방식코드 예시원본 변경
Pass by Value (primitive)update_by_value(arr[3], 10)❌ 복사본만 변경
Pass by Reference (배열)update_by_reference(arr, 3, 20)✅ 원본 변경됨
C에서 int, float, char 등 primitive type은 항상 pass by value!

Structure in C (구조체 — 배열 한계 극복)

서로 다른 타입을 하나로 묶어 관리. typedef로 간편하게 사용.

typedef struct { int year, month, day; } date_of_birth; typedef struct { int student_id; char *name; double grade; date_of_birth dob; // structure within structure! } student; // 값 전달 — 원본 변경 불가 void get_student_value(struct student s) { s.student_id = 20240000; } // 포인터 전달 — 원본 변경 가능 (s->field 로 접근) void get_student_reference(student *s) { s->student_id = 20240000; }
포인터 전달 시 s->field로 접근. 원본 수정 가능!

malloc() — 초기화 없이 할당

// 5개 int = 20 bytes 연속 할당, 초기화 안 됨 (garbage values) int *ptr = (int*) malloc(5 * sizeof(int)); if (ptr != NULL) { // NULL 체크 필수! // ... 사용 ... free(ptr); // 반드시 해제! }
malloc은 초기화 안 함 → garbage values 포함 가능. ptr != NULL 체크 필수!

calloc() — 0으로 초기화하며 할당

// 5개 블록, 각 sizeof(int) = 20 bytes, 모두 0으로 초기화 int *ptr = (int*) calloc(5, sizeof(int));
calloc vs malloc: calloc은 모든 블록을 0으로 초기화. 0 초기화가 필요할 때 사용!

realloc() — 크기 변경

// 기존 20 bytes → 40 bytes로 확장 ptr = realloc(ptr, 10 * sizeof(int)); // 실패 시 NULL 반환, 원본 블록은 그대로 유지됨

free() — 메모리 해제

free(ptr); // 동적 할당 메모리 반드시 해제! (Memory Leak 방지)
동적으로 할당한 메모리는 반드시 free()로 해제. 안 하면 Memory Leak!

Dynamic 2D Array

int row = 3, col = 4; // 행 포인터 배열 할당 int **arr = (int**) malloc(row * sizeof(int*)); // 각 행 할당 for (int i = 0; i < row; i++) arr[i] = (int*) malloc(col * sizeof(int)); // 사용 후 해제 (역순으로!) for (int i = 0; i < row; i++) free(arr[i]); free(arr); // Space/Time Complexity: O(Rows × Columns)

연산설명조건

Array Implementation (배열로 구현)

typedef struct { int top; int capacity; int *array; // char, int, float, double... } Stack; Stack *createStack(int capacity) { Stack *stack = (Stack*) malloc(sizeof(Stack)); if (!stack) return NULL; stack->top = -1; // 빈 스택: top = -1 stack->capacity = capacity; stack->array = (int*) malloc(capacity * sizeof(int)); if (!stack->array) return NULL; return stack; } int isFull(Stack *s) { return s->top == s->capacity - 1; } int isEmpty(Stack *s) { return s->top == -1; } int peek(Stack *s) { return s->array[s->top]; } void push(Stack *s, int item) { if (!isFull(s)) s->array[++s->top] = item; // 먼저 top 증가! else printf("Stack is full!\n"); } int pop(Stack *s) { if (!isEmpty(s)) return s->array[s->top--]; // 반환 후 top 감소! printf("Stack is Empty!\n"); return 0; }
push: ++top 먼저 증가 · pop: top-- 반환 후 감소 · top = -1 이 빈 스택!

Stack 응용 (Applications)

1. Palindrome 체크 (회문 검사)

회문: 앞뒤로 읽어도 같은 단어. 예: mom, level, radar, ABCDCBA

알고리즘:
① 앞 절반(length/2) → 스택에 push
② 홀수 길이면 가운데 문자 skip
③ 나머지 절반 각 문자와 pop 결과 비교 → 다르면 Not Palindrome

int isPalindrome(Stack *stack, char word[], int length) { for (int i = 0; i < length / 2; i++) push(stack, word[i]); int i = length / 2; if (length % 2 != 0) i++; // 홀수: 중간 문자 skip while (i < length) { if (pop(stack) != word[i]) return 0; i++; } return 1; }

3. Postfix Expression (후위 표기식)

표기법설명예
Infix (중위)operand-op-operand (사람용)A + B
Prefix (전위, Polish)op-operand-operand+ A B
Postfix (후위, Compiler)operand-operand-op (컴퓨터용)A B +
Postfix 장점: 컴퓨터가 파싱 쉬움 · 우선순위 불필요 · 괄호 불필요 · 스택으로 효율적 계산

Postfix 계산 알고리즘

① 왼쪽부터 스캔  ② 피연산자 → push  ③ 연산자 → 2개 pop 후 연산  ④ 결과 push

// 예: 5 3 2 * + 8 - // push 5,3,2 → stack: [5, 3, 2] // * → pop 2,3 → 3*2=6 → stack: [5, 6] // + → pop 6,5 → 5+6=11 → stack: [11] // push 8 → stack: [11, 8] // - → pop 8,11→ 11-8=3 → stack: [3] // Result = 3

Infix → Postfix 변환 규칙

• 피연산자 → 출력
• 연산자: 스택 비면 push / top보다 우선순위 높으면 push / 같거나 낮으면 pop
• ( → push  /  ) → ( 만날 때까지 pop
