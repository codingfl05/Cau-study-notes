반사 (Reflection) & 반전 (Inversion)

🪞 반사와 반전이란?

반사(Reflection)는 거울처럼 특정 선/면을 기준으로 뒤집는 변환입니다.
반전(Inversion)은 특정 점을 중심으로 180° 뒤집는 변환입니다.

비유: 반사는 거울 앞에 서는 것, 반전은 점프해서 머리와 발이 바뀌는 것!

2D — x축 반사

x 좌표는 그대로, y 좌표만 부호 반전

x' = x y' = -y 행렬: | 1 0 | | 0 -1 |
💡 거울을 x축에 놓으면, y만 뒤집힌다!

2D — y축 반사

y 좌표는 그대로, x 좌표만 부호 반전

x' = -x y' = y 행렬: | -1 0 | | 0 1 |
💡 거울을 y축에 놓으면, x만 뒤집힌다!

2D — 원점 반전 (Inversion in Origin)

원점을 기준으로 180° 돌리는 것. x, y 모두 부호 반전.

x' = -x y' = -y 행렬 표현: X' = -X = (-I)X 즉: 단위행렬에 -1을 곱한 것!
💡 비유: 지구 반대편으로 터널을 뚫어 순간이동한 것과 같습니다. 모든 좌표가 반대가 됩니다.

2D — 임의 점 P(a,b)에서의 반전

원점이 아닌 점 P(a,b)를 중심으로 반전. 방법: ① P로 평행이동 → ② 원점 반전 → ③ 역이동

T = 평행이동 행렬 (a,b만큼) | 1 0 a | T = | 0 1 b | | 0 0 1 | 반전 행렬: | -1 0 0 | F = | 0 -1 0 | | 0 0 1 | 결합: X' = T · F · T⁻¹ · X
1T⁻¹X: P를 원점으로 이동
2F(T⁻¹X): 원점에서 반전 (-I 곱함)
3T(F(T⁻¹X)): 다시 P 위치로 되돌림

2D — 원점을 지나는 직선 m에 대한 반사

직선 m이 x축과 각도 θ를 이룰 때의 반사. 방법: ① θ만큼 역회전 → ② x축 반사 → ③ 재회전

R = 회전 행렬 (θ): | cosθ -sinθ | R = | sinθ cosθ | 반사 공식: X' = R · | 1 0 | · R⁻¹ · X | 0 -1 |
🔑 핵심 아이디어: 직선을 x축에 겹치도록 회전 → x축 반사 → 다시 원래 각도로 회전

2D — 임의 점 P(a,b)를 지나는 직선 m에 대한 반사

가장 일반적인 경우. 4단계 변환 합성:

1T⁻¹: P를 원점으로 이동
2R⁻¹: 직선 m을 x축에 일치시키도록 역회전
3x축 반사: diag(1,-1,1) 행렬 적용
4R, T: 역순으로 되돌리기
X' = T · R · | 1 0 0 | · R⁻¹ · T⁻¹ · X | 0 -1 0 | | 0 0 1 | 여기서: T = 점 P(a,b)로의 평행이동 R = 각도 θ의 회전 행렬

3D — 좌표 평면에 대한 반사

반사 평면규칙행렬
xy 평면 x'=x, y'=y, z'=-z diag(1, 1, -1)
yz 평면 x'=-x, y'=y, z'=z diag(-1, 1, 1)
xz 평면 x'=x, y'=-y, z'=z diag(1, -1, 1)
💡 패턴: 반사하려는 평면의 법선 축에 해당하는 성분만 -1이 됩니다!

3D — 임의 평면에 대한 반사

점 P(a,b,c)를 지나고 법선 벡터 N을 갖는 평면에 대한 반사.

X' = T · R⁻¹ · F_xy · R · T⁻¹ · X 여기서: T = 점 P(a,b,c)로의 평행이동 R = 법선 벡터 N을 z축으로 이동시키는 회전 F_xy = xy 평면 반사 행렬 (z만 -1)
💡 아이디어: 법선 벡터를 z축으로 "정렬"시키면 → xy평면 반사가 원하는 임의 평면 반사가 됩니다!

3D 회전 (3D Rotations)

🌀 3D 회전의 기초

3D 회전은 각 축을 기준으로 한 회전 행렬로 표현합니다. 반시계 방향(Counter-clockwise)이 양의 방향이며, 오른손 좌표계를 사용합니다.

💡 2D 회전 = 3D에서 z축 기준 회전과 동일!

📐 3가지 기본 회전 행렬

x축 기준 회전 (각도 α)

Rx = | 1 0 0 | | 0 cosα -sinα | | 0 sinα cosα | x좌표는 변하지 않음!

y축 기준 회전 (각도 β)

Ry = | cosβ 0 sinβ | | 0 1 0 | | -sinβ 0 cosβ | y좌표는 변하지 않음! sinβ 위치 주의!

z축 기준 회전 (각도 γ)

Rz = | cosγ -sinγ 0 | | sinγ cosγ 0 | | 0 0 1 | z좌표는 변하지 않음! (2D 회전과 동일)
⚠️ Ry는 sinβ의 부호가 반대! (cosβ 0 sinβ / 0 1 0 / -sinβ 0 cosβ) — 시험에 자주 나옴!
🎯 공통 성질: |Rx| = |Ry| = |Rz| = 1 (모두 행렬식이 1 → 크기 보존)

🔁 임의 축 A 주위 θ 회전 — 5단계 분해

임의 축 A 주위로 회전하려면, A를 좌표 축 중 하나에 먼저 맞추고, 회전 후 되돌리는 방식을 사용합니다.

1z축 기준으로 α 회전: A의 xy평면 투영(Axy)이 x축과 이루는 각도 α만큼 회전. A가 xz 평면에 놓이게 됨 → X' = Rα·X
2y축 기준으로 β 회전: RαA와 x축이 이루는 각도 β만큼 회전. A가 x축과 일치하게 됨 → X' = Rβ·Rα·X
3x축 기준으로 θ 회전: 원하는 회전 적용! → X' = Rθ·Rβ·Rα·X
4y축 기준으로 -β 회전: 단계 2 되돌리기 → X' = Rβ⁻¹·Rθ·Rβ·Rα·X
5z축 기준으로 -α 회전: 단계 1 되돌리기 → X' = Rα⁻¹·Rβ⁻¹·Rθ·Rβ·Rα·X
최종 공식: X' = Rα⁻¹ · Rβ⁻¹ · Rθ · Rβ · Rα · X
💡 비유: 열쇠구멍을 돌리려면 → 열쇠를 수평으로 맞추고(1,2) → 돌리고(3) → 다시 원위치(4,5)

📊 임의 축 회전 — 벡터 분해 공식

점 P를 단위 벡터 A 주위로 θ만큼 회전시킬 때, P를 두 성분으로 분해합니다:

P를 두 성분으로 분해: proj_A P = (A·P)A ← A 방향 성분 (회전해도 안 변함!) perp_A P = P - (A·P)A ← A에 수직인 성분 (이것이 회전됨) 회전 결과: P' = proj_A P' + perp_A P' perp_A P' = (A×P)sinθ + [P - (A·P)A]cosθ 따라서: P' = P·cosθ + (A×P)·sinθ + A(A·P)(1-cosθ)
🎯 이것이 바로 로드리게스 회전 공식(Rodrigues' Rotation Formula)입니다!
💡 A에 평행인 성분은 회전 불변, A에 수직인 성분만 θ만큼 회전. A×P는 수직 방향 단위 벡터 역할.

오일러 각도 (Euler Angles) & 짐벌락

🌐 오일러 각도란?

3D에서 임의의 회전은 세 개의 기본 축 회전의 곱으로 표현할 수 있습니다.

예) Z-Y-X 순서: X' = Rx · Ry · Rz · X 다양한 순서 가능: X-Y-Z, Z-X-Z, Z-X-Y, Y-X-Z ... (12가지 조합)
이름축설명예시
Pitchx축앞뒤로 끄덕이기비행기 기수 올리기/내리기
Rolly축좌우로 기울이기비행기 날개 기울기
Yawz축좌우로 방향 틀기비행기 방향 전환

⚠️ 짐벌 락 (Gimbal Lock)

오일러 각도의 치명적 문제점입니다. 하나의 축 회전이 다른 축의 회전과 겹쳐서 자유도를 잃는 현상입니다.

⚠️ 예시: X-Y-Z 회전에서 Y축을 90° 돌리면 → Z축이 X축과 일치 → 3개의 축이 2개처럼 작동!
즉 3자유도 → 2자유도로 감소!
💡 비유: 조이스틱 두 개가 같은 방향을 가리키게 되어, 아무리 돌려도 특정 방향으로 못 가는 상태

📉 보간 문제 (Interpolation Problem)

두 회전 사이를 부드럽게 보간할 때, 단순히 오일러 각도를 선형 보간하면 경로가 울퉁불퉁해질 수 있습니다. → 쿼터니언이 해결책!

📋 오일러 각도 컨벤션 비교

컨벤션공식특징
Z-X-ZX' = Rz · Rx · Rz · X고전 오일러 각도, 물리학에서 자주 사용
Z-X-YX' = Ry · Rx · Rz · X항공/게임 엔진에서 자주 사용
X-Y-ZX' = Rz · Ry · Rx · X로봇공학에서 자주 사용
⚠️ 행렬 곱셈은 오른쪽부터 적용! X' = Rx·Ry·Rz·X에서 Rz가 먼저 적용됩니다.

쿼터니언 (Quaternion)

💫 왜 쿼터니언인가?

오일러 각도의 짐벌락과 보간 문제를 해결합니다.

특성오일러 각도쿼터니언
표현3개의 각도 (α,β,γ)4-튜플 (w,x,y,z)
짐벌락❌ 발생 가능✅ 발생 안 함
보간❌ 비균일할 수 있음✅ 균일하고 부드러움
직관성✅ 이해하기 쉬움❌ 덜 직관적
계산 효율보통회전 합성 시 효율적

📖 쿼터니언 대수학

윌리엄 해밀턴(William Hamilton)이 복소수의 비가환 일반화로 발명. 4-튜플 w + xi + yj + zk로 표현.

특수 허수 단위의 성질: i² = j² = k² = -1 ij = -ji = k jk = -kj = i ki = -ik = j (마치 3D 공간의 외적처럼 순환!
💡 쿼터니언 q = (w, x, y, z) = (스칼라부분, 벡터부분)

🔄 회전과 쿼터니언 연결

3D 공간의 모든 회전은 하나의 쿼터니언으로 인코딩됩니다.

각 축 기준 단순 회전의 쿼터니언 표현: x축으로 α 회전: qx = (cos(α/2), sin(α/2), 0, 0) y축으로 β 회전: qy = (cos(β/2), 0, sin(β/2), 0) z축으로 γ 회전: qz = (cos(γ/2), 0, 0, sin(γ/2)) 오일러 각도 합성: q = qx · qy · qz ← 쿼터니언 곱셈 → X' = Rx·Ry·Rz·X 와 동일한 결과!
🎯 회전 축 A, 각도 θ로 회전: q = (cos(θ/2), A_x·sin(θ/2), A_y·sin(θ/2), A_z·sin(θ/2))

📐 쿼터니언 → 행렬 변환

쿼터니언 q = (w, x, y, z)를 3×3 회전 행렬로 변환:

R_q = | 1-2y²-2z² 2xy-2wz 2xz+2wy | | 2xy+2wz 1-2x²-2z² 2yz-2wx | | 2xz-2wy 2yz+2wx 1-2x²-2y²|
💡 실제 예시: q = (1,0,0,0) (단위 쿼터니언) → R = 단위행렬 (회전 없음)

🌊 SLERP — 구면 선형 보간

쿼터니언은 4D 단위 구(球) 위의 점으로 표현됩니다. 두 회전 사이 보간 = 구면 위의 호를 따라 이동!

SLERP(q1, q2, t) = q1·sin((1-t)Ω)/sinΩ + q2·sin(tΩ)/sinΩ 여기서 Ω = q1·q2의 사잇각 t = 0: q1 (시작 회전) t = 1: q2 (끝 회전) t = 0.5: 정확히 중간 회전
✅ SLERP 장점: 일정한 각속도로 회전 → 애니메이션이 자연스러움!

3D 직선 & 평면

📏 3D 직선의 표현

3D에서 직선은 한 점 S와 방향 벡터 V로 정의합니다.

직선 방정식: P = S + tV 여기서: S = 시작점 (고정) V = 방향 벡터 t = 매개변수 (t ∈ 실수) t=0: P=S (시작점) t=1: P=S+V t=-1: P=S-V (반대 방향)
💡 비유: 기차가 역(S)에서 출발해 방향(V)으로 달린다. t는 시간!

📐 점과 직선 사이의 거리

점 Q에서 직선(S,V)까지의 거리 d를 구합니다. 핵심: Q-S를 V방향과 수직 방향으로 분해!

d² = |Q-S|² - |proj_V(Q-S)|² proj_V(Q-S) = [(Q-S)·V / |V|²] · V 최종 공식: d = √( |Q-S|² - [(Q-S)·V]² / |V|² )

💡 쉬운 숫자 예시

S=(0,0,0), V=(1,0,0), Q=(2,3,0)이라고 하면:

Q-S = (2,3,0) (Q-S)·V = 2·1 + 3·0 + 0·0 = 2 |V|² = 1 |Q-S|² = 4+9+0 = 13 d = √(13 - 4²/1) = √(13-4) = √9 = 3 → 점 (2,3,0)에서 x축까지의 거리 = 3 ✓

📏 두 직선 사이의 거리

두 직선 P₁=S₁+t₁V₁, P₂=S₂+t₂V₂가 있을 때, |P₁(t₁)-P₂(t₂)|를 최소화합니다.

|P₁-P₂|²를 t₁, t₂에 대해 편미분 → 0으로 놓으면: | V₁·V₁ -V₁·V₂ | |t₁| |(S₂-S₁)·V₁| | V₁·V₂ -V₂·V₂ | |t₂| = |(S₂-S₁)·V₂| 이 2×2 연립방정식을 풀어 t₁, t₂를 구하고 P₁(t₁), P₂(t₂)를 대입하면 최단거리 계산!
⚠️ (V₁·V₂)² - |V₁|²|V₂|² = 0이면 두 직선이 평행! → 다른 방법(한 점에서 거리) 사용

🟡 3D 평면의 표현

3D 평면은 법선 벡터 N과 평면 위의 한 점 P로 정의합니다.

평면 방정식: N·(X - P) = 0 (벡터 형태) ax + by + cz + d = 0 (스칼라 형태) 평면 L = ⟨N, D⟩ 표기: D = -N·P (상수항) L = ⟨N, -N·P⟩
💡 N은 평면에 수직인 방향. 평면 위의 모든 점 X는 N·(X-P)=0을 만족합니다.

📐 점과 평면 사이의 거리

점 Q에서 평면 L=⟨N,D⟩까지의 부호 있는 거리:

d = N·Q - N·P = L·Q 결과 해석: d = 0 → Q가 평면 위에 있음 d > 0 → Q가 법선 방향 쪽(양의 면) d < 0 → Q가 반대 쪽(음의 면)
✅ 부호가 있어서 점이 평면의 어느 쪽에 있는지 바로 알 수 있습니다! (충돌 감지에 유용)

✂️ 직선과 평면의 교점

직선 P₁=S+tV가 평면 N·Q+D=0와 만나는 점을 구합니다.

P₁을 평면 방정식에 대입: N·(S+tV) + D = 0 N·S + N·Vt + D = 0 t = -(N·S + D) / (N·V) t를 구한 뒤 P = S + tV로 교점 계산!
⚠️ N·V = 0이면 직선이 평면과 평행! (교점 없음)

💡 숫자 예시

직선: S=(0,0,0), V=(1,1,1) / 평면: N=(0,0,1), D=-3 (z=3 평면)

N·V = 0·1 + 0·1 + 1·1 = 1 N·S = 0 t = -(0 + (-3)) / 1 = 3 교점: P = (0,0,0) + 3·(1,1,1) = (3,3,3) ✓

✂️ 평면과 평면의 교선

세 평면의 교점

N₁·Q = N₁·P₁ N₂·Q = N₂·P₂ → M·Q = b N₃·Q = N₃·P₃ M = [N₁ᵀ; N₂ᵀ; N₃ᵀ] Q = M⁻¹·b

두 평면의 교선

교선 방향 = N₁ × N₂ (두 법선의 외적) 교선 위의 한 점: Q = [N₁; N₂; N₁×N₂]⁻¹ · [N₁·P₁; N₂·P₂; 0] 교선: P = Q + t(N₁×N₂)

🔄 평면의 변환 (Transforming Planes)

변환 F=TM (M: 선형 변환, T: 이동)을 적용할 때, 평면 L은 어떻게 변환될까?

평면 L = ⟨N, D⟩ 변환 F = TM (M은 3×3 행렬 부분, T는 이동 부분) 변환된 평면: L' = (F⁻¹)ᵀ · L 즉: 점을 F로 변환할 때, 평면은 F의 역행렬의 전치로 변환!
⚠️ 주의: 점은 F로, 평면은 (F⁻¹)ᵀ로 변환. 직관적이지 않지만 이것이 기하학적으로 올바릅니다!
🎯 특히 법선 벡터의 변환에도 동일한 규칙 적용: N' = (M⁻¹)ᵀ·N

📷 투영 (Projection): 3D → 2D

종류특징길이각도평행선
평행 투영 직교 투영, CAD 스케일 변환 가능 보존 안 됨 ✅ 보존
원근 투영 사람 눈, 카메라 스케일 변환 보존 안 됨 ❌ 보존 안 됨

평행 투영 (z=d 평면으로)

| x' | | 1 0 0 0 | | x | | y' | = | 0 1 0 0 | | y | | z' | | 0 0 0 d | | z | | 1 | | 0 0 0 1 | | 1 | 결과: x'=x, y'=y, z'=d

원근 투영 (z=d 평면으로)

유사 삼각형으로 유도: x'/d = x/z → x' = x/(z/d) y'/d = y/z → y' = y/(z/d) 동차 좌표에서: w' = z/d로 나누면 자동 처리! |Rq| = 0이면 z=d 평면에 투영됨
💡 원근 투영에서 멀리 있는 물체(큰 z)는 x/(z/d)이 작아지므로 → 작게 보임. 원근감의 수학적 표현!

🧪 Quiz

Q1
점 (3, 4)를 y축에 대해 반사한 결과는?
✅ 정답! y축 반사는 x→-x, y→y. (3,4) → (-3,4)
Q2
3D 회전 행렬 Ry (y축 기준 β 회전)에서 sinβ의 부호가 다른 위치는?
✅ 정답! Ry = [cosβ 0 sinβ / 0 1 0 / -sinβ 0 cosβ]. Rx, Rz와 달리 부호가 반대 위치에 있습니다!
Q3
로드리게스 회전 공식으로 점 P를 단위 벡터 A 주위로 θ 회전시킬 때 올바른 공식은?
✅ 정답! P를 A에 평행한 성분(불변)과 수직 성분(회전)으로 분해합니다.
Q4
짐벌락(Gimbal Lock)이란?
✅ 정답! 오일러 각도에서 Y축 90° 회전 시 X축과 Z축이 겹쳐 3자유도 → 2자유도가 됩니다.
Q5
쿼터니언에서 x축 기준 90° 회전을 표현하면?
✅ 정답! qx = (cos(α/2), sin(α/2), 0, 0). α=90°이므로 α/2=45°. cos45°=sin45°=√2/2 ≈ 0.707
Q6
점 Q=(2,3,0)에서 직선 S=(0,0,0), V=(1,0,0)까지의 거리는?
✅ 정답! |Q-S|²=13, (Q-S)·V=2, |V|²=1. d=√(13-4)=√9=3. (x축까지 y거리=3)
Q7
평면 N·(X-P)=0에서 점 Q가 "양의 면(positive side)"에 있다는 것은?
✅ 정답! d = N·Q - N·P가 양수이면 법선 벡터 방향쪽(양의 면), 0이면 평면 위, 음수이면 반대편.
Q8
직선 P=S+tV와 평면 N·Q+D=0의 교점을 구하는 t 공식은?
✅ 정답! P=S+tV를 대입: N·S + N·Vt + D = 0 → t = -(N·S+D)/(N·V)
Q9
두 평면의 교선 방향은?
✅ 정답! 교선은 두 평면 모두에 속하므로 양쪽 법선에 수직. N₁×N₂가 두 벡터 모두에 수직입니다.
Q10
변환 F를 적용할 때 평면 L이 변환되는 방식은?
✅ 정답! 평면(법선 벡터 포함)은 역행렬의 전치로 변환됩니다. 이는 법선 벡터 변환 N' = (M⁻¹)ᵀN과 일관됩니다.
